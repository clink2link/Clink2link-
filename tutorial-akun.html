Pengaturan Akun

Panduan mengatur profil, keamanan, dan pembayaran akun Click2Pay.

1. Buka Menu Setting

Masuk ke Dashboard lalu pilih menu Setting untuk mengatur akun kamu.

2. Ubah Data Profil

Kamu dapat memperbarui informasi akun seperti username dan data profil yang tersedia.

3. Ganti Password

Gunakan password yang kuat dan lakukan perubahan secara berkala untuk menjaga keamanan akun.

4. Atur Payment Setting

Tambahkan atau ubah rekening bank maupun e-wallet yang digunakan untuk withdraw.

5. Periksa Informasi Akun

Pastikan seluruh data akun selalu benar agar tidak mengalami masalah saat menggunakan fitur Click2Pay.

Profil

Tips Keamanan Akun

Gunakan password unik, jangan membagikan kode keamanan kepada orang lain, dan selalu pastikan login dilakukan melalui website resmi Click2Pay.

Lindungi Akun Kamu

Jika menemukan aktivitas mencurigakan, segera ubah password dan hubungi support Click2Pay.
